What is a web client?

A web client is the client, or user, side of the web. it typically refers to the Web browser in the user's machine. It may also refer to plug-ins and helper appplications that enhance the browser to support special services from the site. The term may imply the entire user machine or refer to a handheld device that provides Web access.

What is a web server?

What are HTTP and HTTPS?(Which of these protocols would be more appropriate for an online shopping website? Explain the reasoning behind your answer.)

HTTP (HyperText Transfer Protocol): for exchanging text, graphics, audio, video, and other multimedia files on the web.

HTTPS: Encrypted version of HTTP.

For an online shopping website, HTTPS protocol would be more appropriate. Because it is more secure and the communication is encrypted which means the credit card information and other personal information won't be read by others easily.
